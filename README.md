# Miniscope-Calcium-imaging-analysis-code
This repository contains the custom Python analysis pipeline for the manuscript:
"Somatostatin neurons in medial amygdala bidirectionally modulate alcohol-related aggression"
This project investigates how chronic intermittent ethanol intoxication (CIEI) reshapes the activity of somatostatin-expressing (SST+) neurons in the medial amygdala (MeA) and how these changes drive aggressive behavior.
Using in vivo calcium imaging (Miniscope) and automated behavioral tracking (DLC/SimBA), we identify specific neuronal ensembles that encode aggression and track their plasticity throughout alcohol abstinence.
